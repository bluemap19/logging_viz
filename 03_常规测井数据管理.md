## 一、模块概述

### 1.1 模块定位

常规测井数据管理模块负责**标准测井曲线数据**的加载、处理和管理。常规测井是石油地质勘探中最基础、应用最广泛的测井方法，包括自然伽马、电阻率、声波、密度、中子等多种物理参数测量。

### 1.2 核心功能

| 功能 | 说明 |
|------|------|
| **数据加载** | 读取 CSV/XLSX 格式的测井曲线数据 |
| **曲线管理** | 多曲线统一管理和访问 |
| **数据预处理** | 深度对齐、异常值处理、归一化 |
| **数据封装** | 统一的数据访问接口 |

### 1.3 技术特点

- 支持 CSV 和 XLSX 两种文件格式
- 支持多曲线同时加载和管理
- 支持深度范围筛选
- 支持曲线归一化处理
- 懒加载策略，提高内存效率

### 1.4 常见测井曲线类型

| 曲线代码 | 中文名称 | 物理意义 | 单位 | 典型范围 |
|----------|----------|----------|------|----------|
| GR | 自然伽马 | 地层放射性 | API | 0-150 |
| AC | 声波时差 | 声波传播时间 | μs/m | 150-450 |
| DEN | 密度 | 地层体积密度 | g/cm³ | 2.0-3.0 |
| CNL | 中子孔隙度 | 含氢指数 | % | 0-45 |
| RT | 深侧向电阻率 | 地层真电阻率 | Ω·m | 1-1000 |
| RXO | 冲洗带电阻率 | 冲洗带电阻率 | Ω·m | 1-500 |
| SP | 自然电位 | 电化学电位 | mV | -100-100 |
| CAL | 井径 | 井眼直径 | mm | 150-400 |

### 3.3 核心方法详解

#### load_data()

**目的：** 从 CSV 或 XLSX 文件加载测井数据

**原理：**
1. 检测文件扩展名 (.csv 或 .xlsx)
2. 使用相应方法读取文件
3. 提取深度列和曲线列
4. 存储到 `self.data` 和 `self.curves`

**代码示例：**
```python
def load_data(self):
    import pandas as pd
    import os
    
    file_ext = os.path.splitext(self.path)[1].lower()
    
    if file_ext == '.csv':
        self.data = pd.read_csv(self.path)
    elif file_ext in ['.xlsx', '.xls']:
        self.data = pd.read_excel(self.path)
    else:
        raise ValueError(f"Unsupported file format: {file_ext}")
    
    # 提取深度
    if 'DEPTH' in self.data.columns:
        self.depth = self.data['DEPTH'].values
    else:
        # 尝试其他深度列名
        depth_candidates = ['Depth', 'depth', 'MD', 'TVD']
        for col in depth_candidates:
            if col in self.data.columns:
                self.depth = self.data[col].values
                break
    
    # 提取曲线
    self.curve_names = [col for col in self.data.columns 
                        if col.upper() != 'DEPTH']
    
    for curve in self.curve_names:
        self.curves[curve] = self.data[curve].values
```

#### get_data(curve_names, depth_limit, norm)

**目的：** 获取指定曲线和深度范围的数据

**参数：**
- `curve_names`: 要获取的曲线名称列表，None 表示全部
- `depth_limit`: 深度范围 `[depth_min, depth_max]`，None 表示全部
- `norm`: 是否进行归一化

**返回：**
- 包含深度和指定曲线的 DataFrame

**原理：**
1. 根据 curve_names 筛选曲线列
2. 根据 depth_limit 筛选深度行
3. 可选进行归一化处理
4. 返回筛选后的 DataFrame

**代码示例：**
```python
def get_data(self, curve_names=None, depth_limit=None, norm=False):
    # 复制原始数据
    result = self.data.copy()
    
    # 筛选曲线
    if curve_names:
        cols = ['DEPTH'] + [c for c in curve_names if c in self.curve_names]
        result = result[cols]
    
    # 筛选深度
    if depth_limit:
        mask = (result['DEPTH'] >= depth_limit[0]) & \
               (result['DEPTH'] <= depth_limit[1])
        result = result[mask]
    
    # 归一化
    if norm:
        result = self.normalize(curve_names)
    
    return result
```

#### normalize(curve_names)

**目的：** 对测井曲线进行归一化处理

**归一化方法：**
1. **Min-Max 归一化：** 将数据缩放到 [0, 1] 范围
2. **Z-Score 标准化：** 转换为均值为 0、标准差为 1 的分布
3. **自定义范围归一化：** 根据测井曲线的物理范围归一化

**代码示例：**
```python
def normalize(self, curve_names=None, method='minmax'):
    result = self.data.copy()
    
    curves_to_norm = curve_names if curve_names else self.curve_names
    
    if method == 'minmax':
        for curve in curves_to_norm:
            if curve in result.columns:
                min_val = result[curve].min()
                max_val = result[curve].max()
                if max_val > min_val:
                    result[curve] = (result[curve] - min_val) / (max_val - min_val)
    
    elif method == 'zscore':
        for curve in curves_to_norm:
            if curve in result.columns:
                mean = result[curve].mean()
                std = result[curve].std()
                if std > 0:
                    result[curve] = (result[curve] - mean) / std
    
    return result
```

#### filter_curves(method, window)

**目的：** 对测井曲线进行滤波去噪

**滤波方法：**
- **中值滤波 (median)：** 去除尖峰噪声，保持边缘
- **均值滤波 (mean)：** 平滑噪声，但可能模糊边缘
- **高斯滤波 (gaussian)：** 加权平滑，权重呈高斯分布

**代码示例：**
```python
def filter_curves(self, method='median', window=5):
    import numpy as np
    from scipy.ndimage import median_filter, uniform_filter, gaussian_filter1d
    
    result = self.data.copy()
    
    for curve in self.curve_names:
        if curve in result.columns:
            data = result[curve].values
            
            if method == 'median':
                filtered = median_filter(data, size=window)
            elif method == 'mean':
                filtered = uniform_filter(data, size=window)
            elif method == 'gaussian':
                filtered = gaussian_filter1d(data, sigma=window/2)
            else:
                filtered = data
            
            result[curve] = filtered
    
    return result
```

#### remove_invalid_depth(invalid_value)

**目的：** 删除包含无效值的深度点

**代码示例：**
```python
def remove_invalid_depth(self, invalid_value=-999.25):
    import numpy as np
    
    result = self.data.copy()
    
    # 标记无效值
    for curve in self.curve_names:
        if curve in result.columns:
            mask_invalid = result[curve] == invalid_value
            mask_nan = result[curve].isna()
            result.loc[mask_invalid | mask_nan, curve] = np.nan
    
    # 删除所有曲线都有问题的深度点
    result = result.dropna(subset=self.curve_names, how='all')
    
    return result
```

## 八、测井曲线解释基础

### 8.1 岩性识别

| 岩性 | GR | DEN | CNL | AC |
|------|-----|-----|-----|-----|
| 砂岩 | 低 | 中 | 低 | 中 |
| 泥岩 | 高 | 中 | 高 | 高 |
| 石灰岩 | 低 | 高 | 低 | 中 |
| 白云岩 | 低 | 高 | 低 | 低 |

### 8.2 流体识别

| 流体 | RT | DEN | CNL |
|------|-----|-----|-----|
| 水 | 低 | - | - |
| 油 | 高 | - | - |
| 气 | 高 | 低 | 低 |

### 8.3 孔隙度计算

**密度孔隙度：**
```
φ_DEN = (ρ_ma - ρ_b) / (ρ_ma - ρ_f)
```
其中 ρ_ma 为骨架密度，ρ_b 为体积密度，ρ_f 为流体密度

**中子孔隙度：**
```
φ_NL = CNL (经过环境校正)
```

**声波孔隙度 (Wyllie 公式)：**
```
φ_AC = (Δt - Δt_ma) / (Δt_f - Δt_ma)
```

## 十、常见问题

### Q1: 测井曲线存在大量缺失值？

**解决方案：**
1. 检查原始数据质量
2. 使用插值方法填补缺失值
3. 删除缺失严重的深度段

```python
# 线性插值
data['GR'] = data['GR'].interpolate(method='linear')
```

### Q2: 多条测井曲线深度不一致？

**解决方案：**
1. 以一条曲线为基准深度
2. 对其他曲线进行插值对齐

```python
from scipy.interpolate import interp1d

def align_depth(base_depth, base_curve, target_depth, target_curve):
    f = interp1d(target_depth, target_curve, kind='linear', 
                 fill_value='extrapolate')
    return f(base_depth)
```
